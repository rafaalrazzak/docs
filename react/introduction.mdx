## 💡 Apa Itu React?

React adalah pustaka JavaScript yang digunakan untuk membangun antarmuka pengguna (UI). Bayangkan React seperti LEGO, di mana kamu bisa membuat **komponen kecil** dan menggabungkannya untuk membentuk aplikasi yang besar. Komponen-komponen ini bisa diatur sedemikian rupa agar responsif dan dinamis.

## 🌟 Kenapa React Keren?

Berikut beberapa alasan mengapa React sangat populer di kalangan pengembang:

* **Mudah Dipelajari**: Konsep dasar React sederhana dan mudah dipahami.

* **Fleksibel**: Kamu bisa membangun berbagai jenis aplikasi, mulai dari aplikasi web sederhana hingga aplikasi besar.

* **Populer**: Banyak perusahaan teknologi besar menggunakan React, jadi ini skill yang sangat berharga.

## 🛠 Persiapan Proyek Pertama

Sebelum mulai, pastikan kamu sudah menginstal **Bun.js** atau **Node.js** di komputermu.

### Install Bun.js atau Node.js:

* **Bun.js**: Bun adalah alternatif modern untuk menjalankan JavaScript, lebih cepat dan ringan.Instalasi Bun.js:

```bash
  curl -fsSL [https://bun.sh/install](https://bun.sh/install) | bash
```

* **Node.js**: Jika lebih familiar dengan Node.js, kamu bisa mengunduhnya di [Node.js Official Site](https://nodejs.org/).

Setelah menginstal, buat proyek baru dengan menggunakan Bun.js atau Node.js.

## 📁 Struktur Folder Penting

Di dalam proyek React, kamu akan menemui beberapa folder dan file penting:

* **`src/`**: Tempat kode sumber aplikasi React-mu.

* **`public/`**: Tempat file statis seperti gambar atau ikon.

* **`package.json`**: Menyimpan daftar dependensi dan script proyek.

## ✨ Gaya Penulisan di React

Di React, ada beberapa gaya penulisan yang perlu kamu ketahui untuk membuat kode lebih konsisten:

### 1. Camel Case 🐪

Untuk nama variabel atau properti, kita menggunakan gaya penulisan camel case.
Contoh: `namaDepan`, `tombolKlik`.

### 2. Pascal Case 🏷️

Untuk nama komponen, kita menggunakan Pascal Case.
Contoh: `KartuProfil`, `TombolLike`.

### 3. Kebab Case 💅

Untuk nama class CSS, kita menggunakan kebab case.
Contoh: `kartu-profil`.

## 🔧 Komponen React: Dasar Utama

Komponen adalah blok penyusun aplikasi React. Komponen dapat berupa fungsi yang menerima **props** (data) dan menampilkan UI.

Contoh komponen sederhana:

```jsx
function ProfilPengguna({ nama, foto }) {
  return (
    <div>
      <img src={foto} alt={nama} />
      <h3>{nama}</h3>
    </div>
  );
}
```

## 🎛 State: Membuat Komponen Interaktif

State adalah cara untuk membuat komponen React berinteraksi. Dengan state, komponen bisa berubah berdasarkan aksi pengguna.

Contoh penggunaan state untuk tombol like:

```jsx
import { useState } from 'react';

function TombolLike() {
  const [jumlahLike, setJumlahLike] = useState(0);

  return (
    <button onClick={() => setJumlahLike(jumlahLike + 1)}>
      👍 Like: {jumlahLike}
    </button>
  );
}
```

## 🌟 Rendering Bersyarat

Tampilan aplikasi bisa berubah sesuai kondisi. Berikut contoh penggunaan rendering bersyarat di React:

```jsx
function PesanLogin({ isLogin }) {
  return <div>{isLogin ? "Selamat Datang!" : "Silakan Login."}</div>;
}
```

## 🍃 Tambahan: Styling dengan Tailwind CSS

Untuk mempercantik tampilan aplikasi, kamu bisa menggunakan **Tailwind CSS**, framework CSS yang sangat populer.

### Install Tailwind CSS:

<Steps>
  <Step title="Install Tailwind dengan Bun atau npm:">
    ```
       bun i -D tailwindcss postcss autoprefixer
       npx tailwindcss init -p
    ```
  </Step>

  <Step title="Tambahkan konfigurasi Tailwind di file index.css">
    ```
       @tailwind base;
       @tailwind components;
       @tailwind utilities;
    ```
  </Step>

  <Step title="Gunakan kelas Tailwind di komponen React">
    ```
    function App() {
      return (
        <div className="flex justify-center items-center h-screen bg-gray-100">
          <h1 className="text-4xl font-bold text-blue-600">
            React itu Keren!
          </h1>
        </div>
      );
    }
    ```
  </Step>
</Steps>

Dengan Tailwind CSS, kamu bisa membuat antarmuka yang responsif dan modern dengan cepat!
